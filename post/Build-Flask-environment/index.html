搭建Flask环境

2020-04-06 #学习笔记

前言

在搭建falsk环境之前，你的系统上必须装有python环境。

开始安装

如果你使用windows系统，建议Win+R输入powershell，使用powershell进行以下操作。

安装virtualenv

在命令行输入pip install virtualenv即可安装virtualenv。

该命令需要 管理员权限，如果你使用的是windows系统，请使用管理员方式打开powershell，如果你是Linux/Mac os可在pip命令前添加sudo，如果你使用的是ubuntu，可以使用包管理器进行安装：sudo apt-get install virtualenv

安装虚拟环境

请使用cd命令切换至程序所在的文件夹
然后在该文件夹下执行virtualenv venv以安装 venv 。

激活虚拟环境

Linux/Mac os请使用venv/bin/activate命令
windows环境请使用venv\\scripts\\activate命令

注意，若windows环境下出现

venv\scripts\activate : 无法加载文件 D:\...\venv\scripts\activate.ps1，因为在此系统上禁止运行脚本
。有关详细信息，请参阅 https:/go.microsoft.com/fwlink/?LinkID=135170 中的 about_Execution_Policies。
所在位置 行:1 字符: 1
+ venv\scripts\activate
+ ~~~~~~~~~~~~~~~~~~~~~
    + CategoryInfo          : SecurityError: (:) []，PSSecurityException
    + FullyQualifiedErrorId : UnauthorizedAccess

那么

  1. 请你以管理员身份打开powershell
  2. 在命令行输入set-executionpolicy remotesigned
  3. 然后将看见
    执行策略更改
 执行策略可帮助你防止执行不信任的脚本。更改执行策略可能会产生安全风险，如 https://go.microsoft.com/fwlink/?LinkID=135170
 中的 about_Execution_Policies 帮助主题所述。是否要更改执行策略?
 [Y] 是(Y)  [A] 全是(A)  [N] 否(N)  [L] 全否(L)  [S] 暂停(S)  [?] 帮助 (默认值为“N”):
  4. 输入A即可

最后

到这里，flask的环境就已经搭建完成，可以开始你的表演了！
