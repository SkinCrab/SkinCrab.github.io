DeepinV20 | 系统安装

2020-04-21 #实用教程 #学习笔记

Windows玩腻了？来玩玩 深度 操作系统吧！

这篇文章将教你安装Deepin20Beta + Windows 10双系统。
（鉴于Deepin20版本还不太稳定，故不建议在生产环境中使用）

安装前

准备

在安装之前你需要准备

  1. 一个大于8GB的空U盘（用于制作启动盘
  2. 大于20GB的硬盘分区
  3. Deepin镜像
  4. Deepin启动盘制作工具

磁盘分区

  1. 右键单击Windows徽标，选择磁盘管理
  2. 在你欲安装deepin的分区上右键，选择 压缩卷
  3. 然后填入压缩空间量（建议剩余空间>20GB）
    计算公式：压缩空间量=可压缩空间大小-剩余空间大小
  4. 分区完毕

制作启动盘

  1. 打开所下载的Deepin启动盘制作工具
  2. 选择你所下载的Deepin镜像
  3. 然后选择你的U盘

    4.等待制作完成即可

Start

Windows系统注意事项

  1. 若你的系统是Windows，请关闭快速启动，具体操作方法点这里
  2. 若你的系统是Windows，建议进入BIOS，关闭UEFI.

开始安装

  1. 首先你得关机
  2. 关机后进入启动项选择菜单（我的联想是F12，不同厂商不一样，详见文末帮助）,选择U盘启动
  3. 选择使用语言（建议使用母语）
  4. 选择手动安装
  5. 选择你所分出来的区
    5.1 若出现提示，选择左边的按钮（忘记截图了）
    注意：除非你知道在干什么，否则请勿选择右边的，那会格式化全盘！
  6. 按照下图进行配置即可
  7. 点击新建，等待安装完成。

进入系统

在初次进入系统时一般来说会让你填写用户名以及密码，按要求做就行了。

Enjoy your Deepin journey！

使用体验

个人感觉DeepinV20相较V1X版本而言，颜值提高了不少，但鉴于是Beta版本，很多地方优化的还不够好，再加上个人电脑硬件比较菜，使用体验略有欠缺。

帮助

启动项选择菜单

  • 台式机一般为：Delete 键
  • 笔记本一般为：F2 键
  • 惠普笔记本一般为：F10 键
  • 联想笔记本一般为：F12 键
  • 苹果电脑为：C 键
